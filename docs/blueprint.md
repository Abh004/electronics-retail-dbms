# **App Name**: ElectroPos

## Core Features:

- Product Management: CRUD operations for managing products, including name, price, stock, and brand association.
- Customer Management: CRUD operations for managing customer data, including personal details and contact information.
- Employee Management: CRUD operations for managing employee data, including personal details, designation and hire date.
- Order Processing: Create and manage orders, automatically calculating totals. Backend is governed by triggers and functions for automatic total amount, payment, balance calculations and stock management.
- Payment Processing: Process payments for orders, updating the order balance and payment history.
- Dashboard Summaries: Provide a dashboard showing insights into the business operation and calling the total spent API.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to convey trust and stability in financial transactions.
- Background color: Light gray (#F9FAFB) for a clean and modern interface.
- Accent color: Electric Purple (#7E57C2) to highlight interactive elements and important actions.
- Headline font: 'Space Grotesk', a sans-serif, will be used to provide a techy, computerized look
- Body font: 'Inter', a sans-serif, providing a clean, modern look for readability
- Use a set of consistent, professional icons from a library like FontAwesome or Material Icons.
- Implement a responsive layout using Tailwind CSS grid and flexbox for optimal viewing on different devices.
- Incorporate subtle transitions and animations using Tailwind CSS transitions to enhance user experience.
